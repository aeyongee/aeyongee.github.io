## 쿠버네티스, 인프라 환경 관리를 코드로 했을 때 장점
1. 인프라에 대한 History 관리가 편해짐
2. 인프라 작업에 대한 추적 가능
3. 인프라 환경별 파일 생성
    - 시간 있을 때 미리 구성 가능
    - 실제 작업시에는 복붙
4. 인프라 반복 작업이 줄어들고, 퀄리티 향상에 집중 가능
5. 새 인프라 작업시 이전 경험이 반영된 코드 활용 가능

<br>

### 느낀점
회사에서도 서버 세팅 가이드가 있지만, 이슈 발생시 조치한 내용이 기록되어있지 않아 어려움을 겪었던 적이 많은데,          
이처럼 쿠버네티스로 세팅하고 코드로 관리하면 이력 관리에도 좋고 해보지 않은 사람도 접근하기가 좋을 것 같다고 느꼈다.
